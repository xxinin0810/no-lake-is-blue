# 没有湖泊是蓝色的

一个 WebVR 叙事体验，讲述关于记忆、时间与水的故事。

改编自同名剧本。六个场景已完成，更多还在做。

---

## 技术栈

- Three.js + WebXR（VR 模式）
- Vite + TypeScript
- 自定义 GLSL Shader（黑白/彩色后处理、湖面着色器）
- 全部视觉用程序化几何体生成，不依赖外部 3D 模型

## 运行方式

```bash
npm install
npm run dev
```

然后打开 http://localhost:5173/

## VR 模式

用 PICO 4 / Quest 浏览器打开（或者桌面 Chrome + WebXR emulation 插件），点击右下角 VR 按钮进入。
